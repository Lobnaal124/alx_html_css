# Headphones Website

A simple and elegant website for the Headphones company, showcasing their services, results, and providing a contact form. The website includes animated sections for a more engaging user experience.

## Project Overview

This project is a static website created using HTML and CSS. It contains the following sections:

1. **What we do**: Describes the services offered by the company.
2. **Our results**: Showcases the company's achievements with a visual representation.
3. **Contact us**: A form for users to reach out to the company.
4. **Footer**: Includes company logo and social media links.

## Technologies Used

- HTML
- CSS

## Features

- Responsive design that works well on both desktop and mobile devices.
- Animations on hover for interactive experience.
- Clean and modern design.
- Easy to customize and extend.
