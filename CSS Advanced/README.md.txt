Welcome to my project! This project demonstrates my skills in HTML, CSS, and basic web development concepts.

## Project Structure

The project directory contains the following files:

- `index.html`: The main homepage of the portfolio.
- `about.html`: About page with information on the TechCon 2024 conference.
- `schedule.html`: Schedule page with a detailed timetable of events.
- `register.html`: Registration page with a user-friendly form.
- `contact.html`: Contact page with contact details and a form.
- `styles.css`: CSS file for styling the pages.
- `images/`: Directory containing images used in the project.
